Detailed Description

Namespace containing compile-time flags for application configuration.

This namespace defines compile-time constants that control specific behavior of the application, such as verbosity and output redirection. These flags are set based on predefined macros and allow for conditional compilation.

Controls whether standard output should be redirected.

The __f__redirect flag is set to true if __FLAG_APP_REDIRECT_STDOUT__ is defined at compile-time; otherwise, it is set to false.
