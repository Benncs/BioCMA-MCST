\doxysubsection{Detailed Description}
Holds the data required to execute a simulation case. 

The {\ttfamily \mbox{\hyperlink{structCaseData}{Case\+Data}}} struct encapsulates all the necessary components needed to perform a simulation. It includes the simulation unit, parameters, transitioner, and execution information. This structure is typically used to manage and transfer simulation data between different stages or components of the simulation process.
