\doxysubsection{Detailed Description}
A data structure representing the state of a container in the simulation. 

The {\ttfamily \mbox{\hyperlink{structMC_1_1ContainerState}{Container\+State}}} struct stores information about the liquid and gas volumes, the number of cells, a unique identifier, and a span of concentrations. It is aligned to the cache line size specified by {\ttfamily Exec\+Info\+::cache\+\_\+line\+\_\+size}.
