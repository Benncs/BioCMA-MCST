\doxysubsection{Detailed Description}
Represents the spatial domain where Monte Carlo particles can exist. 

The {\ttfamily \mbox{\hyperlink{classMC_1_1ReactorDomain}{Reactor\+Domain}}} class models a spatial domain using a compartment-\/based approach. Each compartment within the domain holds and manages its own state through an instance of {\ttfamily \mbox{\hyperlink{structMC_1_1ContainerState}{Container\+State}}}. The compartments collectively represent the entire spatial domain in which particles interact, move, and evolve according to Monte Carlo simulations.

The {\ttfamily \mbox{\hyperlink{classMC_1_1ReactorDomain}{Reactor\+Domain}}} class is responsible for maintaining the structure and state of the compartments, and for providing access to the compartments for simulation and modeling purposes.
