17// * @brief A basic wrapper around Kokkos View to handle non-fixed size containers
18// * with an API compatible with `std::vector`.
19// *
20// * This class provides a dynamic array-like container built on top of Kokkos
21// * View, allowing it to manage memory in a parallel-friendly way while offering
22// * an interface similar to the standard C++ `std::vector`. It supports
23// * operations like resizing, inserting elements, and migrating data between
24// * different memory spaces.

26// * Like all other dynamic container, the real amount of allocated memory is
27// * bigger than one requested by user to avoid reallocating memory when a single
28// * element is pushed into vector
